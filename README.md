## PurCarte Mod

基于 [komari-theme-purcarte](https://github.com/Montia37/komari-theme-purcarte) 修改的 [Komari](https://github.com/komari-monitor/komari) 个人用实用向主题。

## 修改总结

- 移除多视图模式，统一为精简表格布局，并对实例与详情页面重新排版。
- 统计条与节点列表的交互与溢出处理改进。
- 图表体验优化（负载/延迟），含刻度排版、提示与视觉反馈提升。
- 统一数值、日期与单位格式（3 位有效数字、ISO 风格时间、Mbps 显示）。
- 新增运行时长与到期时间的颜色标识。
- 移除浅色主题，默认字体更换为 Fira Sans Condensed，并完善字体资源。
- 本地化与文案优化，并补齐多处标签与说明。
